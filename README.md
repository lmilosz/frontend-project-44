# игры разума 

## Установка

```
make install
```

## Как начать играть?

### 1 игра - проверка на четность
Описание: 
игроку в течение 3-х раундов будет выдаваться случайное число. От игрока требуется ответить yes, если оно четное, и no если число нечетное.

чтобы запустить игру:
```
make brain-even
```
